# Product Images Directory

Place all product images in this directory.

## Image Naming Convention

Images should be named according to the product handle and color:

- `cb-star-sweatshirt-navy.jpg`
- `cb-star-hoodie-black.jpg`
- `the-wilson-sneaker-red.jpg`
- etc.

## Supported Formats

- JPG/JPEG
- PNG
- WebP

## Image Requirements

- Recommended size: 800x800px minimum
- Square aspect ratio preferred
- High quality (for product detail pages)
