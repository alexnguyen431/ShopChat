## Instructions

1. Upload your images to the `/public/images/` directory
2. Name each image file according to the mapping above
3. The product titles in the site will automatically match the images based on the handle

## Notes

- All image paths in `lib/mockData.ts` are set to `/images/[filename]`
- Next.js will automatically serve images from the `/public` directory
- Make sure image file names match exactly (case-sensitive)
